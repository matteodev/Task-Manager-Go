<b><h1>Task Manager develop with Go [DEMO]</h1></b>
<p>This is a simple task manager application developed with Go. It allows you to create, edit, complete, and delete tasks. The tasks are stored in a JSON file.</p>

<b><h2>Features</h2></b>
<ul>
    <li>Create new tasks with a title and description.</li>
    <li>Edit existing tasks.</li>
    <li>Mark tasks as complete.</li>
    <li>Delete tasks with a confirmation modal.</li>
</ul>
<p>I have used the flash cookie to store the alert messages.</p>
